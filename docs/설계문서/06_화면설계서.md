# 화면 설계서 — BaroFarm(바로팜)

> Vue 3 SPA · Vue Router 기준 **총 29개 화면** · 접근권한: 공개 / 로그인 / 판매자(SELLER) / 관리자(ADMIN)
> 근거: `frontend/src/router/index.js` + `frontend/src/views/*.vue`

## 2. 공통 UI / 네비게이션
- **헤더**: 로고·검색·카테고리 플라이아웃·5탭(홈/About/베스트/특가/혜택및공지)·장바구니·알림·로그인 상태. **모바일에서는 카테고리 버튼을 탭 상단으로 세로 재배치**(겹침 방지).
- **라우트 가드**(`router.beforeEach`): `meta.auth`(로그인) / `meta.seller`(판매자) / `meta.admin`(관리자) 미충족 시 로그인 또는 차단.
- **상태관리**(Pinia): auth · cart · wishlist · follow · notification.
- **디자인**: 그린 테마 scoped CSS, 썸네일은 자체호스팅 이미지(`/seed-images`) + 이모지 폴백.
